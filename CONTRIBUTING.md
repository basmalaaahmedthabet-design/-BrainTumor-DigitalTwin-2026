## 🤝 Types of Collaboration Welcome

### ✅ We Welcome
- **Bug reports** in the demo interface
- **Documentation improvements** and translations
- **Clinical validation data** sharing agreements (via formal MoU)
- **Research collaborations** with academic institutions
- **Deployment partnerships** for LMIC healthcare settings

### ❌ Not Accepted via GitHub
- Pull requests to the core clinical implementation (proprietary)
- Requests to add or change model architecture (IP-sensitive)
- Commercial use inquiries (contact author directly)

---

## 📋 How to Contribute

### 1. Reporting Issues
Use the GitHub Issues tab for:
- Bug reports in the demo interface
- Documentation errors
- Broken links or references

Please include:
- Your Python version and OS
- Steps to reproduce the issue
- Expected vs actual behavior

### 2. Academic Collaboration
For research partnerships:
1. Email basmalaa.thabet@medicalai.edu with:
   - Your institutional affiliation
   - Description of your research interest
   - Proposed collaboration timeline
2. Author will respond within 5-7 business days
3. Formal research agreement required before data sharing

### 3. Clinical Deployment
For hospital/clinic deployment partnerships:
1. Contact author with institutional credentials
2. Provide evidence of IRB/ethics approval at your institution
3. Formal MoU required before deployment access

## 📜 Code of Conduct

All contributors must:
- Respect intellectual property rights
- Not attempt to reverse-engineer proprietary components
- Not use contributions to circumvent the CC BY-NC-SA 4.0 license
- Maintain patient privacy in any shared data (de-identification required)
- Acknowledge the original author in any derivative works
